# awesome-design（2019-12-03）
> 给独立开发者，开箱即用、免费漂亮的设计资源

### 写在前面

作为一名设计专业的学渣，毕业后，由于按捺不住一颗躁动的心，一直希望不受约束地做自己喜爱的产品。

我开始自学编程，初尝了写一个[小程序](https://github.com/junbin123/likeday)的快感。

一手包办了产品的方向、设计、交互、原型、开发。

虽然，「我的产品我做主」的感觉很爽，但有时要做的事情太多，不免要分心，把时间放在了不重要的事上。

于是，我整理这份开箱即用的设计资源，帮助那些乐于折腾的小伙伴创造 side project。

他们几乎都是免费；同时，大部分资源我自己也实践过；所以实际上，你看的，是我大浪淘沙过的。

如果你是设计小白，推荐阅读这篇文章：[七个实用的设计技巧](https://medium.com/refactoring-ui/7-practical-tips-for-cheating-at-design-40c736799886)。它非常易懂，了解这些技巧，你可以马上用在你的项目上。

最后，欢迎各位小伙伴提交 issue，分享自己用过的好东西。

我会持续更新，感谢 star 鼓励！
